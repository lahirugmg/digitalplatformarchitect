# IMP-020: Interactive module framework

**Status:** Proposed
**Priority:** High
**Category:** Platform UX
**Effort:** Medium
**Created:** 2026-02-15

## Description

Create a reusable framework for interactive learning modules with steps, toggles, and overlays.

## Proposed Enhancements

- Standard layout for module header, controls, canvas, and notes
- Step-by-step navigation controls
- Module metadata (difficulty, time, prerequisites)

## Benefits

- Consistent UX across learning modules
- Faster development of new interactive experiences

## Acceptance Criteria

- New modules follow a shared layout template
- Steps can be configured from data
- Controls are keyboard accessible

## Estimated Effort

- Medium (2-3 weeks)
