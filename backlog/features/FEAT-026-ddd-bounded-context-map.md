# FEAT-026: DDD bounded context map explorer

**Status:** Proposed
**Priority:** Medium
**Category:** Interactive Learning
**Effort:** Medium
**Created:** 2026-02-15

## Overview

Interactive bounded context map that highlights domain areas and context relationships.

## User Story

**As an** architect
**I want** to explore bounded contexts
**So that** I can align architecture with the business domain

## Interactive Behavior

- Highlight a context (Shipping, Inventory, Billing)
- Show context mapping relationships
- Toggle integration patterns between contexts

## Acceptance Criteria

- Context boundaries are clearly visible
- Relationships are labeled
- Users can focus on a single domain
